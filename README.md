# Rock Paper Scissors

## Rules

* Rock beats scissors
* Scisors beats paper
* Paper beats Rock

## get started

1. Install gulp globally `npm install -g gulp`
2. clone or download repository
3. `npm install`
4. `npm start`
5. `gulp sass`
6. Open index.html in browser
