# Rock Paper Scissors

## rules

* Rock beats scissors
* Scissors beats paper
* Paper beats rock

## get started

1. Install gulp globally `npm install -g gulp`
2. Clone or download repository
3. `npm install`
4. `gulp sass`
5. Open index.html in browser
